# 🚀 Старт: твой AI-ассистент — HRBP по подписке

Эта папка — твоё рабочее пространство с Claude Code. Здесь ИИ хранит контекст о твоей роли, метриках и о том, как ему с тобой работать. Все скиллы уже внутри — ничего ставить отдельно не нужно.

На всё ~40-60 минут, из них 16 минут — тест профиля.

---

## Что понадобится

- Windows или macOS
- **Cursor** (бесплатная версия) — редактор со встроенным терминалом, в нём будем работать
- **Git** — чтобы скачать эту папку
- Доступ к Claude: **API-ключ от Сержа** — он пришлёт его тебе лично (покупать подписку не нужно). Ключ — строка вида `sk-ant-api03-...`

---

## Шаг 1. Установить Cursor (5 мин)

1. Скачай с [cursor.com](https://cursor.com) → Download
2. Поставь как обычную программу, при первом запуске можно всё пропускать (Skip) — платная подписка Cursor НЕ нужна, работаем на бесплатной
3. Открой Cursor. Терминал внутри: меню **Terminal → New Terminal** (или `` Ctrl+` ``)

Все команды ниже вводятся в этот терминал.

## Шаг 2. Установить Git (3 мин)

- **Windows:** скачай с [git-scm.com](https://git-scm.com/downloads), ставь со всеми настройками по умолчанию (Next-Next-Next)
- **macOS:** в терминале набери `git --version` — если не установлен, система сама предложит поставить

После установки на Windows — **перезапусти Cursor** (иначе терминал не увидит git).

## Шаг 3. Установить Node.js (3 мин)

Проверь в терминале:
```
node --version
```
Если видишь `v20.x` или выше — пропускай шаг. Если нет — скачай **LTS-версию** с [nodejs.org](https://nodejs.org), поставь как обычную программу, перезапусти Cursor.

## Шаг 4. Установить Claude Code (2 мин)

```
npm install -g @anthropic-ai/claude-code
```
Проверь:
```
claude --version
```

## Шаг 5. Скачать рабочую папку (2 мин)

В терминале:
```
git clone https://github.com/sergevalovoi/recruittech-ai-workspace.git
```
Затем в Cursor: **File → Open Folder** → выбери папку `recruittech-ai-workspace`. Открой терминал заново (`` Ctrl+` ``).

(Если git clone не сработал — запасной путь: открой ссылку в браузере → зелёная кнопка **Code** → **Download ZIP** → распакуй → открой папку в Cursor.)

## Шаг 6. Подключить API-ключ (5 мин)

Ключ тебе пришлёт Серж лично. Сохрани его в переменную окружения — тогда Claude Code подхватит его сам и не будет просить логин.

**Windows** — в терминале Cursor:
```
setx ANTHROPIC_API_KEY "sk-ant-api03-ТВОЙ-КЛЮЧ"
```
После этого **полностью перезапусти Cursor** (иначе переменная не подхватится).

**macOS** — в терминале:
```
echo 'export ANTHROPIC_API_KEY="sk-ant-api03-ТВОЙ-КЛЮЧ"' >> ~/.zshrc
source ~/.zshrc
```

Проверь — запусти в папке проекта:
```
claude
```
Если Claude Code всё равно предлагает войти через браузер (OAuth) — переменная не подхватилась: перезапусти Cursor/терминал и проверь, что ключ вставлен без лишних пробелов.

Ключ никому не пересылай и никуда не выкладывай.

## Шаг 7. Пройти тест профиля — «тиндер» (16 мин)

Открой файл `tools/behavior-profile-tool.html` **двойным кликом в проводнике** (откроется в браузере, интернет не нужен).

Это форс-чойс тест по 15 моделям поведения: выбираешь, что больше про тебя. По итогу получишь свой профиль + рассчитанный профиль твоего AI-ассистента (где он усиливает тебя, где компенсирует). Можно подвинуть ползунки, если с чем-то не согласна.

В конце нажми **Экспорт JSON** — сохрани файл, он нужен на следующем шаге.

## Шаг 8. Пройти онбординг (15 мин)

В терминале, внутри Claude:
```
/team-onboarding
```
Claude спросит про роль, метрики и задачи, покажет 5 обязательных правил работы с AI, затем попросит JSON из теста. По итогу создаст:
- `COMPANY.md` — твоя роль и правила
- `AGENT-PERSONALITY.md` — как ассистент ведёт себя именно с тобой

Готово. Со следующего запуска `claude` читает эти файлы сам.

## Что уже в папке

```
recruittech-ai-workspace/
├── README-START.md            ← ты здесь
├── CLAUDE.md                  ← инструкция для Claude, что читать при старте
├── COMPANY.md                 ← создаст онбординг
├── AGENT-PERSONALITY.md       ← создаст онбординг
├── tools/
│   └── behavior-profile-tool.html   ← тест профиля («тиндер»)
├── .claude/
│   ├── settings.json          ← безопасные permissions
│   └── skills/                ← 8 скиллов, все уже установлены
└── notes/                     ← сюда пишут /daily-log и /reflect
```

## Твои первые команды после онбординга

- `/daily-log` — вечерний чек-ин: что сделано по метрикам, план на завтра (3-5 мин)
- `/reflect` — раз в неделю: факт vs план, что оставляем, что меняем
- Рекрутинг: `/candidate-research` (ресёрч кандидата), `/eval-screening` (оценка подборки резюме)
- Продажи: `/lead-research` (ресёрч лида перед консультацией), `/polish` (доводка текста/КП)
- `/lightresearch` — быстрый поиск фактов с источниками

## Если что-то не так

- `claude` не найден → `npm install -g @anthropic-ai/claude-code` заново, перезапусти Cursor
- Просит войти заново → внутри claude набери `/login`
- Пишет про usage limit → не поломка: у подписки лимит на 5-часовое окно, восстановится сам, контекст не потеряется
- Git/Node не видны после установки → перезапусти Cursor
- Всё остальное → пиши Сержу

## Правила безопасности

- API-ключ (если он у тебя) — никогда не пересылать в чатах и не коммитить в git
- Данные кандидатов и клиентов — только по whitelist-правилу (онбординг покажет 5 правил AI Mandate)
- Эта папка — твоя личная, не расшаривай её наружу
