# Рабочее пространство сотрудника — HRBP по подписке

## Startup
В начале каждой сессии прочитай эти файлы для полного контекста:
- `COMPANY.md` — моя роль, метрики, обязательные правила работы с AI
- `AGENT-PERSONALITY.md` — как именно ты должен вести себя со мной (под мой поведенческий профиль)
- `metrics-tracker.md` (если есть) — мои метрики: план и факт
- `notes/` — журнал по дням и неделям (читай последние записи при необходимости, не все)

Если `COMPANY.md` / `AGENT-PERSONALITY.md` отсутствуют или содержат только плейсхолдер — онбординг ещё не пройден. Предложи запустить `/team-onboarding` прежде чем делать что-либо ещё.

## Правила
- Веди себя как описано в `AGENT-PERSONALITY.md` — это не декорация, а рабочая инструкция
- Все 5 правил AI Mandate из `COMPANY.md` — обязательны без исключений, независимо от задачи
- Персональные данные кандидатов и клиентов — только по whitelist-правилу из AI Mandate. Если не уверен, можно ли грузить данные в AI — значит нельзя, спроси у Сержа

## Маршрутизация моделей (cost discipline)
По умолчанию — самая дешёвая модель, способная справиться с задачей. Переключение через `/model`.
- **Haiku** — поиск по файлам, чтение, сортировка, форматирование, простые правки
- **Sonnet** — дефолт: тексты, анализ, ресёрч, подготовка материалов
- **Opus** — только по явной просьбе «думай глубже» и для финального ревью важных материалов

Если задача выглядит большой (много файлов, много итераций, 30+ минут непрерывной работы) — сначала скажи оценку, потом начинай.

## Скиллы в этой папке
Все скиллы предустановлены в `.claude/skills/` — ничего ставить не нужно, вызываются командой `/имя`:

| Скилл | Что делает | Для кого в первую очередь |
|---|---|---|
| `/team-onboarding` | Первичная настройка: роль + профиль поведения → COMPANY.md и AGENT-PERSONALITY.md | Все, один раз |
| `/daily-log` | Ежедневный чек-ин: факт vs план по твоим метрикам, стрик | Все, каждый день |
| `/reflect` | Еженедельное ретро: дельта недели, keep/revert, фокус | Все, раз в неделю |
| `/polish` | Доводка текста до 9/10 через цикл Generator→Evaluator | Тексты, КП, презентации |
| `/candidate-research` | Ресёрч кандидата перед интервью/оффером | Рекрутинг |
| `/eval-screening` | Оценка резюме/подборок против критериев вакансии, батчем | Рекрутинг |
| `/lead-research` | Ресёрч компании-лида перед первичной консультацией | Продажи |
| `/lightresearch` | Быстрый поиск фактов с источниками (15-20 агентов, не 100+) | Все |

Ролевые скиллы (рекрутинг/продажи) определяются на онбординге и фиксируются в `COMPANY.md` — не навязывай чужие.

## Заметки
`notes/` — сюда пишут `/daily-log` и `/reflect`. Личные заметки пользователя тоже здесь.

## Длинная память
Эти файлы — твоя память между сессиями. Узнал новое о пользователе, его задачах или предпочтениях — предложи обновить `COMPANY.md` / `AGENT-PERSONALITY.md`. Не забывай то, что узнал — записывай.
